Navigating the Latent Space of Generative Models Using Paired Comparisons

Deep Learning/Convex Optimization | Fall 2019- Spring 2020

Our algorithm, Paired Comparisons, provides an interface for users to use their natural intuition to describe complicated models. This project was created as a means for users to navigate the latent space of generative models, in particular variational-autoencoders (VAEs). We provide a library allowing users to localize a point in the latent space of any given generative model, via an interface that allows users to compare between two sampled points in space.

Check out my team's Google Colab where we experiment our technique on the MNIST digit set.

And here's the GitHub Wiki, further explaining all our cool work!

Electrical Power Grid Contingency Analysis Verification

Graph Theory/Python | Summer 2019

In this project I utilized the Python library NetworkX to convert a database of power device information into an undirected multigraph. Using an edge-contraction method, I linked all device nodes to its respective protecting device (i.e. circuit breaker), and output a dataset containing a set of all contingencies for any possible outage. This automated the monthly process of creating hand-made contingencies, reducing a multiple-day process to 10 minutes!
